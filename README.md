# Gun Accessories Store

A lightweight, static storefront demo for showcasing gun accessories. The project is pure HTML/CSS/JS with no build step, making it easy to open locally and explore.

## Features

- Product listing and product detail flows
- Cart and checkout pages
- Responsive layout and shared styling
- Simple client-side state with vanilla JavaScript

## Pages

- `index.html` - Landing page
- `products.html` - Product catalog
- `cart.html` - Shopping cart
- `checkout.html` - Checkout form

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in your browser.

No dependencies or build tools are required.
